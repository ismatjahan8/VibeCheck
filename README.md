# VibeCheck

Chat application (FastAPI + React/Vite) designed for Railway deployment.

## Tech
- Backend: FastAPI, SQLAlchemy, Alembic, Postgres, WebSockets (Redis pub/sub)
- Frontend: React, TypeScript, Vite, Tailwind

## Local dev (high level)
- Backend: `uvicorn app.main:app --reload`
- Frontend: `npm run dev`
